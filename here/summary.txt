Salary Calculator 

Panoramica
- Prodotto: small web app per stimare il netto mensile/annuo a partire dalla RAL (Retribuzione Annua Lorda).
- Scopo: fornire ai dipendenti o ai team HR uno strumento immediato per valutare il netto a seguito di contributi e imposte.

Punti di forza
- Interfaccia semplice e intuitiva: input minimo (RAL) e risultati rapidi con breakdown delle voci (INPS, IRPEF, addizionali).
- Generazione PDF client-side: possibilità di scaricare il report in PDF direttamente dal browser.
- Architettura leggera: progetto costruito con tecnologie moderne che garantiscono performance e facilità di integrazione.

Tecnologie utilizzate
- Astro (v5.x): framework statico/ibrido per siti veloci, usato come base del progetto per pagine performanti e build statiche o server-side.
- Componente front-end: `src/components/SalaryCalculator.astro` contiene la UI e la logica di interazione (input, controllo, rendering risultati).
- JavaScript client-side per calcoli: la logica di calcolo (aliquote semplificate, contributi, IRPEF progressiva) è implementata direttamente nel componente per semplicità e reattività.
- Generazione PDF: integrazione client-side con `html2pdf.js` caricata dinamicamente da CDN — permette di convertire l'area risultati in PDF senza dipendenze server.
- Stili globali minimi: `src/styles/global.css` contiene un reset leggero e le regole CSS essenziali (input, button, layout del calcolatore). Gli stili sono centralizzati nel layout (`src/layouts/Layout.astro`) per una manutenzione semplice.
- Build & dev: script standard `npm run dev` e `npm run build` (Astro). Il progetto usa `package.json` con dipendenza ad `astro`.

Perché scegliere questa soluzione
- Rapidità di prototipazione: Astro consente di ottenere una pagina funzionante con bundle minimo e ottime performance out-of-the-box.
- Facile integrazione: il componente è standalone e può essere incluso in pagine esistenti (es. portali HR, intranet aziendali o come micro-widget).
- Privacy e costi bassi: tutti i calcoli e la generazione PDF avvengono client-side, evitando costi server e problematiche di gestione dati sensibili.

Come provare il progetto (quick start)
1) Aprire terminale nella cartella `here` del repository.
2) Eseguire:
   npm install
   npm run dev
3) Aprire il browser all'indirizzo mostrato (es. `http://localhost:3000`), inserire la RAL e cliccare `Calcola`. Usare `Scarica PDF` per ottenere il report.
